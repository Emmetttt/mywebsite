Maths Identities and General Solutions

Taylor Expansion


  • $$f(x) = f(x_0) + f'(x_0)x + \frac{f''(x_0)}{2!}x^2...$$
  • $$\frac{1}{1-x} = 1 + x + x^2 + x^3$$
  • $$cosx = 1 - \frac{1}{2}x^2 + {1}{24}x^4$$
  • $$sinx = x - {1}{6}x^3 + {1}{120}x^5$$
  • $$tanx = x + \frac{1}{3}x^3 + \frac{2}{5}x^5$$
  • $$e^x = 1 + x + \frac{1}{2}x^2$$
  • $$ln(1+x) = x - \frac{1}{2}x^2 + \frac{1}{3}x^3$$


Trig Identities


  • $$\sin^2x + \cos^2x = 1$$
  • $$\tan^2x + 1 = \sec^2x$$
  • $$\tan{x} = \frac{\sin{x}}{\cos{x}}$$
  • $$\sin{x \pm y} = \sin{x} \cos{y} \pm \cos{x} \sin{y}$$
  • $$\cos{x \pm y} = \cos{x} \cos{y} \mp \sin{x} \sin{y}$$
  • $$\tan{x \pm y} = \frac{\tan{x} \pm \tan{y}}{1 \mp \tan{x} \tan{y}}$$
  • $$\sin(2{x}) = 2 \sin{x} \cos{x}$$
  • $$\cos{2x} = \cos^2{x} - \sin^2{x}$$
  • $$\sin^2x = \frac{1-\cos(2x)}{2}$$
  • $$\cos^2x = \frac{1+\cos(2x)}{2}$$
  • $$\tan^2x = \frac{1-\cos(2x)}{1+\cos(2x)}$$


Calculus


  • Product Rule $$\frac{d}{dx}(fg) = \frac{df}{dx} g + f \frac{dg}{dx}$$
  • Quotient Rule $$\frac{d}{dx}\frac{f}{g} = \frac{f'g - g'f}{g^2}$$
  • Chain Rule $$\frac{df}{dx} = \frac{dg}{dx}\cdot\frac{df}{dg}$$
  • Integration by parts $$\int f g' dx = fg - \int g f' dx$$
  • Integration by Subsitution $$\int f(x) dx = \int f(x(g))x'(g)dg$$
